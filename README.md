# worq
tools to navigate project, currently only support wezterm

Development:
```bash
bun run index.ts
```

Usage:
- compile
  ```bash
  bun build index.ts --compile --outfile worq
  ```
- run
  ```bash
  ./worq
  ```
- adding directory
  ```bash
  ./worq --add
  ```
- removing directory
  ```bash
  ./worq --remove
  ```
